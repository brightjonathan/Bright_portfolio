Project details

  • The Expense Tracker application which will keep a track of Income-Expense of a user on a day to day basis.
  • This application takes Income and amount from user and divides it into two which are the income-expense and it also displays a user balance.
  • The Expense tracker takes in a positive amount as an income and negative amount as an expense which help the user differentiate between income and Expense.
  • Developed an api that stores income, expense and balance in the database.
  • Used React in the frontend,Node, Express.
